Deploy in minutes

  1. 1) Push this repo to GitHub (public or private).
  2. 2) In Render, create a new Web Service from the repo.
  3. 3) When prompted, set Root Directory to dashboard.
  4. 4) Render will detect Python; keep build: pip install -r requirements.txt.
  5. 5) Set start: gunicorn app.main:app.
  6. 6) Optional env vars: APP_NAME, APP_TAGLINE, PORT.

Local run: cd dashboard && pip install -r requirements.txt && flask --app app run
